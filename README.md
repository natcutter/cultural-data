# cultural-data
This repository contains all data subsets used to create visualisations for the book Cultural Data: The Intimate Analytics of Digital Collections, by Tyne Daile Sumner, Nat Cutter, and Rachel Fensham.

The full versions of the Australian Cultural Datasets can be found at https://www.acd-engine.org/datasets and https://drive.google.com/drive/folders/1KONv8_3OScXKwEakPnQ2ZgfcXBPYuz3z.
