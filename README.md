# cultural-data
This repository contains all data subsets used to create visualisations for the book Cultural Data: The Intimate Analytics of Digital Collections, by Tyne Daile Sumner, Nat Cutter, and Rachel Fensham.

The repository is organised by chapter, so certain datasets are duplicated.

**Discrete datasets** used for specific analyses, tables, and visualisations in this book

Cutter, Nat. ‘KMN Alternative Names Dataset’. March 2025.

Cutter, Nat, Rachel Fensham, and Tyne Daile Sumner. ‘KMN Dataset’. March 2025.

East, Scott, Joanna Mendelssohn, Aneshka Mora, Ivy Zheng, and ACD-Engine Team. ‘DAAO Selected Biographical Dataset’. November 2022.

Fensham, Rachel. ‘KMN Collectives Dataset’. March 2025.

Muñoz, Justin. ‘KMN DAAO AustLit Works Dataset’. March 2025.

Muñoz, Justin. ‘KMN DAAO Exhibitions Dataset’. March 2025.

Muñoz, Justin, and Sean Bussutil. ‘Archibald Prize All Participants by Year Dataset’. 2022.

Stewart, Kristin, and Gillian Butcher. ‘Circus Oz Dataset’. July 2023.

**The Australian Cultural Dataset** as a whole, which is used in certain visualisations, is available through the ACD-Engine website, https://www.acd-engine.org/datasets, and through Google Drive, https://drive.google.com/drive/folders/1KONv8_3OScXKwEakPnQ2ZgfcXBPYuz3z.
Zheng, Ivy, Justin Munoz, David Carlin, Scott East, Chris Hay, Joanna Mendelssohn, John Macarthur, David McMeekin, Deborah van der Plaat, and the ACD-Engine Team. ‘Australian Cultural Dataset’. Australian Cultural Data Engine. August 2023.
